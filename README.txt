== Description ==

A plugin that lets customers quickly place orders with a order form layout. Built with React to provide a great user experience.
Filter products by category and then simply add them to the cart. Increase sales and conversions!

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `quick-order.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
